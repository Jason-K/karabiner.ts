# Definitions Reorganization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reorganize `src/definitions/` into trigger-based files plus an `apps/` subfolder while keeping `karabiner-output.json` byte-identical to the pre-reorg baseline.

**Architecture:** Pure structural refactor across 14 files. No engine, data, or rule-semantic changes. Each task is a self-contained step with a deterministic verification: rebuild and diff `karabiner-output.json` against a baseline snapshot. Any non-empty diff blocks the task.

**Tech Stack:** TypeScript, tsx, karabiner.ts library, `npm run check` pipeline (typecheck + lint + tsx test runner + build).

**Spec:** `docs/superpowers/specs/2026-05-20-definitions-reorg-design.md`

import type { TapHoldConfig } from "../engine";
import { singleKeyTapHoldMappings } from "./single-key";
import { hyperTapHoldMappings, buildHyperLauncherRules, hyperLauncherMappings } from "./hyper";
import { leftCommandTapHoldMappings, buildLeftCommandRule, buildCmdQRule } from "./left-command";
import { rightOptionTapHoldMappings, buildRightOptionLauncherRules, rightOptionLaunchers } from "./right-option";
import { antinoteTapHoldMappings, buildAntinoteRules } from "./apps/antinote";

function mergeTapHoldRecords(
  ...records: Array<Record<string, TapHoldConfig>>
): Record<string, TapHoldConfig> {
  const merged: Record<string, TapHoldConfig> = {};
  for (const record of records) {
    for (const [key, value] of Object.entries(record)) {
      if (key in merged) {
        throw new Error(`Duplicate tapHold key across definition files: ${key}`);
      }
      merged[key] = value;
    }
  }
  return merged;
}
